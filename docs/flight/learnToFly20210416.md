## 三下开关

辅助燃油泵——开车时使用

空速管加温——冬天气温低（<10℃）且有明显水汽（降水、云中）时使用

室外温度通过 PFD 左下角 OAT 指示

## 磁电机

| 开关位置  | 描述                   |
| --------- | ---------------------- |
| Both      | 飞行中常用             |
| L 或 R 位 | 试车、发动机抖动时使用 |

## 驾驶盘

左手把盘，右手油门

左手**食指**按报话按钮

前推杆叫**稳杆**
后拉杆叫**带杆**

> _AHRS_ 指 姿态航向基准系统

## 音频

手持麦克风是在 报话按钮有问题时使用的，作为备用

## 燃油选择活门的使用流程

1. 开 辅助燃油泵
2. 观察 燃油流量
3. 操作 燃油选择活门

## 其它仪表

计时器的开始计时时机：滑油压力 >20 psi

ADF 控制盒第三个按钮用于切换频率

ADF 控制盒旋钮的调值方法：

| 数值位 | 操作方法       |
| ------ | -------------- |
| 百位   | 大旋钮         |
| 十位   | 小旋钮         |
| 个位   | 小旋钮拔出再旋 |
